# Ausbringmenge Rechner

Eine kleine Web-App zur Berechnung der maximalen Ausbringmenge aus Strecke, Arbeitsbreite und Tankvolumen.

## Version

Aktueller Stand: v0.5

- Hochformat für die installierte Handy-App festgelegt.
- Standortknopf direkt in die Karte unter die Zoom-Tasten verschoben.
- Eigener Standort wird bei aktivierter Freigabe als Punkt auf der Karte angezeigt.

## GitHub Pages

1. Neues GitHub-Repository erstellen, zum Beispiel `ausbringmenge-rechner`.
2. Diese Dateien hochladen:
   - `index.html`
   - `manifest.json`
   - `service-worker.js`
   - `icon.svg`
   - `README.md`
3. Im Repository `Settings` > `Pages` öffnen.
4. `Deploy from a branch` wählen.
5. Branch `main` und Ordner `/root` auswählen.
6. Speichern und die angezeigte HTTPS-Adresse am Handy öffnen.

In Chrome kann die App danach über das Menü zum Startbildschirm hinzugefügt werden.
